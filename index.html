about open 5g-empower

5G-EmPOWER is an open toolkit for SDN/NFV research and experimentation in mobile networks. Its flexible architecture and the high-level programming APIs allow for fast prototyping of novel services and applications.

5G-EmPOWER blurs the line between radio and core network introducing the concept of Programmable Data Plane which abstracts the radio and packet processing resources available in a network.

5G-EmPOWER natively support multi-tenancy allowing MVNOs to specify RRM policies separately for each slice while ensuring performance isolation between and efficient spectrum utilization.
