use cases

radio slicing

5G-EmPOWER allows MVNOs to deploy custom resource allocation schemes within their network slice. The underlying system enforces strict performance isolation between MVNOs and ensures efficient resource utilization across the network in spite of the non-deterministic nature of the wireless medium.

mac offloading

In 5G-EmPOWER MAC resources are virtualized making it possible to offload part of them as LVNFs. Notice how, only higher MAC functions can be deployed as LVNF, while the physical layer and the lower MAC are not virtualized.
